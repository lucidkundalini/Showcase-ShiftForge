# Showcase Contract Notes

This repository contains selected backend source files, not a full runnable application.

The public contract being shown is:

1. system/runtime questions can be answered from deterministic runtime truth
2. traces carry provider/model/path metadata
3. provider and MCP introspection can be exposed safely without leaking secrets
4. live behavior must be distinguished from intended direction

The original private repository contains the executable test suites used to verify these behaviors.
